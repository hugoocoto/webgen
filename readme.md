# Web Gen

Generate html static webpages from ctml. 

## ctml

This is a html clone using C macros

## File structure

- `mainpage.ctml`: Main content
- `head.ctml `: Webpage head
- `header.ctml`: Metadata and info
- `index.ctml`: The index list 

- `build.c`: Embeddable c script
- `webgen.c`: Include files and call build.
